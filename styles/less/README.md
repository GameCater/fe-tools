## ijijin_builder/stylebuild
- pc：PC端
- mobile：移动端

其中：
- 功能(.f-) 常用样式的使用，通常这些选择器具有固定样式表现，比如清除浮动等；
- 元件(.u-) 组件，不涉及到外部布局，比如按钮、icon等；
- 模块(.m-) 一个语义化的较大的整体，通常与业务相关，比如导航、登录、列表等；
- 布局(.g-) 外部布局相关样式的整体，比如相对定位；
- 状态(.z-) 不同状态展示不同形态；
- 动画(.a-) 动画类；
- JS操作(.j-) 涉及到JS操作的节点。
