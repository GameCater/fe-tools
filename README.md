## * Chrome插件FE-Tools

### 功能：
- 工具网站搜索（`√`）
- 本地收藏夹网站搜索（`√`）
- CSS属性/Moo-CSS搜索（`√`）
- url转二维码及svg矢量图（`√`）
- 图片压缩及转base64（`√`）
- px/rem/vw换算计算器（`√`）
- rgb/hsb/hex色值换算（`√`）
- 中英翻译
- 工具函数库搜索
- 搜索自定义拓展

[前往>>](https://github.com/MichealWayne/fe-tools/tree/master/chrome-extension)

### 安装方式

clone本仓库到本地，点击“加载已解压的拓展程序”选择本地仓库目录下的chrome-extension目录。
